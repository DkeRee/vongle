Hey! Thanks for checking out my platformer, Vongle! As you can see, I'm not too great at making names for my games, but hey, at least it's original. Vongle is a pixel art styled platformer that runs in Snap!, which is a block code IDE similar to Scratch. I wish that I could've made it in pure JavaScript but I had to in order for it to be qualified for my school's coding competition. I won second place in the competition and the prize is a large rainbow owl as shown on the left.
